🛠️ Link Obfuscation Tool (Developer Only)

Convert direct download URLs (ROMs) into Base64 code before adding them to your games.json to keep them hidden from search bots.

Structure sample in games.json:

{
  "title": "Game Title",
  "system": "snes", // snes, nes, gba, gbc, genesis, etc.
  "downloadUrl": "YOUR_BASE64_CODE_HERE",
  "coverUrl": "https://link-to-cover.png"
}
